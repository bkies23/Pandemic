# Pandemic
This program tracks Covid-19 data from March 1, 2020 through the most recent completed month. Users can add, delete, or modify rows and then view the data with charts and tables. The data is stored in a MySQL database which I created. Designing this program helped me develop MySQL skills as it uses Java's ResultSet to store, retrieve, and manipulate data from the Covid-19 table. In particular, after a User modifies data in a row or deletes and inserts a new row, using @incementVaue in a PreparedStatement updates the entire table instantly. If you create a folder to download the Pandemic.jar into, you can then extract it there and the Covid19 tables should open in the current directory.  
The program uses Covid-19 data from the New York Times and Washington Post Covid Trackers.
